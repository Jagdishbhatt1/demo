# NAGP 2026 Technology Band III Batch on Kubernetes & DevOps Advance + FinOps (Cloud) assignment

## Overview
#### This project demonstrates the deployment of a multi-tier application on Kubernetes consisting of:

* Service API Tier (Spring Boot Microservice)
* Database Tier (MySQL Database)
* The application exposes REST APIs externally through Kubernetes Ingress and retrieves data from a MySQL database running inside the cluster.
